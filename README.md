# AI Character App

这是一个展示AI角色的Flutter应用程序。

## 功能特性

### 首页 (Home Screen)
- 以网格视图展示所有AI角色
- 支持搜索功能（可按角色名称、角色类型和描述搜索）
- 下拉刷新更新角色列表
- 角色卡片展示：
  - 角色头像
  - 昵称
  - 角色类型
  - 描述简介

### 待实现功能
- 角色详情页面
- 收藏功能
- 分类筛选

## 使用说明

1. 首页浏览：
   - 打开应用后自动加载所有角色
   - 下拉可刷新角色列表
   
2. 搜索功能：
   - 点击顶部搜索图标进入搜索模式
   - 可输入关键词搜索角色
   - 支持模糊搜索
   - 点击清除按钮可重置搜索
